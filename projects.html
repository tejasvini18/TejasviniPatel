PROJECTS

Stories where quality changed outcomes.

Each project is a case study in how structured QA, automation, and security thinking can transform delivery.

Cloth Shopping – E‑Commerce Platform

End‑to‑end testing of a full e‑commerce shopping experience, ensuring reliability across UI, API, and payment systems.

  • Tested product listing, filtering, cart, checkout, and order lifecycle.
  • Performed cross‑browser and responsive testing.
  • Validated payment gateway flows and order confirmations.

Food Ordering System

Multi‑vendor food ordering platform with real‑time order tracking and payment workflows.

  • Tested menu, cart, delivery, and cancellation flows.
  • Performed API testing for order status updates.
  • Validated concurrency and high‑traffic ordering scenarios.

Pocket Wallet – Digital Payments

Secure wallet system supporting transfers, bill payments, and transaction history.

  • Tested wallet top‑up, transfers, refunds, and OTP flows.
  • Performed security testing (XSS, CSRF, SQLi).
  • Validated encryption and transaction ledger accuracy.

Library Management System

Complete LMS for book inventory, student management, and issue/return workflows.

  • Tested search, reservation, fine calculation, and overdue logic.
  • Performed DB validation for book availability and audit logs.
  • Designed regression suite for every release cycle.

E‑Learning System

Learning platform supporting courses, quizzes, assignments, and progress tracking.

  • Tested enrollment, video playback, quiz engine, and certificates.
  • Performed API testing for scoring and progress tracking.
  • Validated role‑based access and exam load stability.

DNS System Testing

Validation of DNS resolution, propagation, and record accuracy across global servers.

  • Tested A, AAAA, CNAME, MX, TXT, and NS records.
  • Validated TTL behaviour and caching logic.
  • Performed DNSSEC and negative testing.

Trademark Search & Verification

Platform for searching, validating, and tracking trademark applications.

  • Tested search accuracy across trademark classes.
  • Validated application workflows and document uploads.
  • Performed API testing for indexing and classification.

WHOIS Lookup System

System for retrieving domain ownership, registrar, and expiry details.

  • Tested WHOIS queries for multiple TLDs.
  • Validated rate‑limiting, throttling, and error handling.
  • Ensured accuracy of registrar and expiry information.

HMRS – Hospital Management System

Healthcare platform for patient records, appointments, billing, and staff management.

  • Tested EMR accuracy, billing, and appointment workflows.
  • Performed PHI security testing and access control validation.
  • Designed regression suite for critical medical modules.

Campus Management System

System for managing students, staff, attendance, fees, and academic records.

  • Tested admissions, attendance, timetable, and fee modules.
  • Validated role‑based access for Admin, Teacher, Student, Parent.
  • Performed DB validation for academic records.
