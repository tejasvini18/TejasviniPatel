Projects & Case Studies

Product‑level QA, automation, and security work across multiple domains.

Cloth Shopping Platform

End‑to‑end QA for a fashion e‑commerce platform including product listing, filters, cart, checkout, and order history. Designed regression suites and automated critical flows using Playwright, ensuring stable releases during frequent campaign updates and seasonal sales.

Performed cross‑browser testing, payment gateway validation, and discount logic verification to prevent revenue‑impacting defects.

E‑Learning Portal

Tested a modern e‑learning platform with course catalog, enrolment, video streaming, quizzes, and certificates. Validated role‑based access for students, instructors, and admins, ensuring secure content access and accurate progress tracking.

Designed API tests for course, user, and progress endpoints and executed performance checks on peak‑time usage scenarios.

Food Ordering Application

QA and automation for a food ordering system covering restaurant selection, menu browsing, cart, coupon application, and real‑time order status. Focused on reliability of checkout and payment flows under high‑traffic periods such as weekends and evenings.

Implemented API tests for order creation, status updates, and delivery tracking, and validated integration with third‑party payment providers.

LMS – Library Management System

Tested a library management system with book search, reservations, lending, returns, fines, and admin inventory management. Ensured data integrity across transactions using SQL validation and boundary value analysis on due dates and fine calculations.

Created structured test cases and regression packs to support ongoing feature enhancements and database migrations.

Pocket Wallet & Digital Payments

QA for a digital wallet solution supporting balance top‑ups, peer‑to‑peer transfers, transaction history, and security controls such as OTP and device binding. Performed negative testing on transaction limits, failed payments, and concurrency scenarios.

Applied security testing techniques to identify potential data exposure and ensured compliance with secure handling of sensitive financial information.
