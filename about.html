About Me

A quality engineering specialist with deep expertise in automation, security, and enterprise‑grade testing.

I am a Senior QA Engineer & SDET with extensive experience in Manual Testing, Automation Engineering, API Testing, Performance Testing, and Security Testing, delivering high‑quality software across e‑commerce, financial, government, and security‑critical platforms.

I specialise in designing and developing enterprise‑grade automation frameworks using Playwright, Selenium, Cypress, TestNG, Cucumber/BDD, and Page Object Model (POM). These frameworks have improved test coverage, reduced regression cycles, and enabled teams to release faster with higher confidence.

Beyond automation, I bring strong expertise in API testing, performance engineering, and security testing. I have executed JMeter load tests simulating real‑world peak traffic, uncovering backend bottlenecks and improving page load speeds by up to 25%. I have also performed penetration testing using OWASP ZAP, Burp Suite, Nmap, and OWASP Top 10 methodologies, identifying vulnerabilities such as SQL Injection (SQLi), XSS, DNS spoofing, and data exposure before production.

I don’t just test software — I engineer reliability, prevent defects, and protect business‑critical workflows. By embedding quality early in the SDLC and STLC and working closely with developers, BAs, and product owners, I consistently reduce defect leakage and improve release confidence.

✔ Automation Framework Development: Playwright, Selenium, Cypress, TestNG, POM

✔ API & Microservices Testing: Postman, RestAssured, REST, JSON, Swagger

✔ Performance Engineering: JMeter, LoadRunner, bottleneck analysis

✔ Security Testing: OWASP Top 10, Burp Suite, OWASP ZAP, Nmap

✔ CI/CD & DevOps: Jenkins, Git, GitHub, Azure DevOps, Maven

✔ Agile Delivery: JIRA, Confluence, Sprint planning, standups, retrospectives

✔ Documentation & Leadership: Test strategy, test plans, mentoring, UAT

With a strong technical foundation, hands‑on experience across multiple domains, and a focus on precision and structure, I ensure that quality is not just tested at the end — it is engineered into every stage of the product lifecycle.
